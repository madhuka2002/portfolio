## Introduction

**Pizza Mania** is a feature-rich smart pizza ordering app developed collaboratively with my team. It focuses on providing a personalized customer experience while offering efficient tools for admin management. The app allows users to interact with a **chatbot for ordering and support**, create **custom pizzas**, and track their orders in real-time. Integrated **location detection via Google Maps** and multiple payment options ensure a smooth and modern food ordering experience.

## Features

### Customer Features
- Interactive chatbot for ordering and support 🤖
- Smart location detection with Google Maps 🗺️
- Custom pizza builder with ingredient selection 🍕
- Favorites, cart, and checkout with COD & Stripe 💳
- Order history, real-time tracking, and delivery updates 🚚

### Admin Features
- Order management with status updates and driver assignment
- Menu management across multiple branches
- User management: block, suspend, activate users
- Driver management and content updates via SQLite ✏️
- Sidebar options to manage About & Policy pages and delivery drivers 🚗

## Tech Stack

- **IDE**: Android Studio
- **Language**: Java
- **Databases**: Firebase (Cloud) + SQLite (Local)
- **Payments**: Stripe API Integration
