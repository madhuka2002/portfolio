## Introduction

Jetwin LMS is a Moodle-based Learning Management System (LMS) developed and deployed to meet the specific requirements of an educational institution. Hosted on **DigitalOcean**, it provides a reliable and scalable environment for delivering online courses, managing academic content, and supporting both students and educators.

## Features

- **Course Management**: Create, organize, and manage courses with ease.
- **User Management**: Role-based access for administrators, teachers, and students.
- **Assessment Tools**: Online quizzes, assignments, and grading functionality.
- **Secure Hosting**: Deployed on DigitalOcean for high availability and performance.

## Tech Stack

- **Platform**: Moodle
- **Hosting**: DigitalOcean Droplet (Ubuntu)
- **Web Server**: Apache/Nginx
- **Domain**: [lms.jetwin.edu.lk](https://lms.jetwin.edu.lk)

## Deployment Overview

- Configured and deployed on a DigitalOcean server with LAMP stack.
- Domain setup with SSL for secure HTTPS access.
- File and database backups implemented for data protection.
- Performance optimization done via Moodle caching and server tuning.
